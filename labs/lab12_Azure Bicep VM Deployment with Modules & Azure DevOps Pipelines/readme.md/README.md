# 🚀 AZ-400 DevOps Engineer Portfolio

A collection of hands-on Azure DevOps and Cloud Engineering labs completed while preparing for the **AZ-400: Designing and Implementing Microsoft DevOps Solutions** certification.

## 🎯 What This Portfolio Demonstrates

* Azure DevOps Administration
* CI/CD Pipeline Implementation
* GitHub Actions Automation
* Docker Containerization
* Azure App Service Deployments
* Azure Key Vault Integration
* Infrastructure as Code (Bicep)
* Azure Load Testing
* Agile Planning with Azure Boards
* YAML Pipeline Development
